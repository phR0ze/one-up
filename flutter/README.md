### NixOS Dev Env
1. [Launch VSCode for Flutter UI](../README.md#vscode-for-flutter-ui)

2. Build and run the flutter UI locally simply press `F5`

3. Flutter might need a clean rebuild in some cases, run:
   ```bash
   $ cd ~/Projects/oneup/flutter
   $ flutter clean
   $ flutter build linux
   ```
